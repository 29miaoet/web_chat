# Web Chat

## Info
This github project was created due to the lack of readily available and reliable 
anonymous chat services. It can be used when requiring sign in or authentication 
is either too time-consuming or improbable. All data is stored in a Supabase project 
hosted [here](https://supabase.com/dashboard/project/jlpwjhnqgkpfleuqrtwr), it can 
be iframed in any html webpage and used in any desktop device.

## Attributes

- ✅Functional UI and chat interface
- ✅Polished interface on desktop devices
- ⚠️There does not exist any mobile support as of yet
